# IBM Data Analyst Capstone Project: Stack Overflow Developer Survey Analysis

## Project Overview

This project was completed as part of the IBM Data Analyst Professional Certificate. It analyzes the Stack Overflow Developer Survey dataset to identify trends in programming languages, databases, platforms, and developer demographics.

The project follows the complete data analysis workflow, including data collection, cleaning, exploratory data analysis, visualization, and dashboard creation to generate meaningful insights from real-world developer survey data.

---

## Objectives

The main objectives of this project are to:

- Collect and import the Stack Overflow Developer Survey dataset.
- Clean and prepare the data for analysis.
- Perform exploratory data analysis (EDA).
- Visualize developer trends using interactive charts.
- Identify the most popular technologies used by developers.
- Build an interactive dashboard to communicate findings.

---

## Dataset

The dataset used in this project is based on the **Stack Overflow Developer Survey**, one of the world's largest annual surveys of software developers.

The dataset contains information such as:

- Programming Languages
- Databases
- Web Frameworks
- Platforms
- Developer Roles
- Employment Status
- Education
- Age
- Country
- Years of Coding Experience

## Project Workflow

### 1. Data Collection

- Imported the Stack Overflow survey dataset.
- Loaded CSV files into Pandas DataFrames.

### 2. Data Wrangling

- Removed duplicates.
- Handled missing values.
- Converted data types.
- Cleaned text fields.
- Prepared the dataset for analysis.

### 3. Exploratory Data Analysis

Performed statistical analysis including:

- Summary statistics
- Frequency distributions
- Correlation analysis
- Outlier detection

### 4. Data Visualization

Created visualizations including:

- Bar Charts
- Histograms
- Scatter Plots
- Pie Charts
- Box Plots
- Heatmaps
- Interactive Plotly Charts

### 5. Dashboard Development

Built an interactive dashboard displaying:

- Top Programming Languages
- Most Popular Databases
- Most Used Platforms
- Developer Demographics
- Technology Trends
